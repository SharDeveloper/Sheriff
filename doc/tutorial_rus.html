Назначение

Модуль предназначен для разбора конфигурационных файлов в формате Sheriff. (SHar Easy to Read confIg File Format)

Функции

loadFromFile(fileName STD::String) <>STD::Dict - функция загружает конфигурационный файл с указанным именем, если файл не удалось загрузить, то функция возвращает nothing.
loadFromString(config STD::String) STD::Dict - функция загружает конфигурационный файл из строки.

Если файл с конфигурацией не является валидным utf-8 файлом, то выкидывается исключение с кодом "const::STD::eid_invalidCharacter"
В случае некорректности конфигурации, обе функции выкидывают исключение, данные в исключения - массив из номера строки и номера столбца при разборе которых произошла ошибка.

Возможные ошибки:
Код ошибки Значение
const::Sherif::eid_notExpectedToken найдена последовательность символов, которая не должна быть в том месте где она находится
const::STD::eid_outOfRange значение ключа выходит за границы допустимых значений
const::Sherif::eid_alreadyExist для нового ключа указано имя уже существующего
const::Sherif::eid_notExpectedEOL парсер неожиданно достиг конца строки
const::Sherif::eid_invalidName неверное имя опции или категории
const::Sherif::eid_incorrectEscapeSequence неверная управляющая последовательность в строке

При успешном разборе конфигурации, функции возвращает объект типа STD::Dict.
В качестве ключа используется либо имя опции (без категории), либо имя категории окружённое кавычками [].

Если в качестве ключа используется имя опции, то значение соответствующее ключу может иметь следующие типы:
Тип Подробности
STD::Bool соответствует значениям on, yes, true, off, no, false
STD::Int соответствует десятичным и шестнадцатеричным числам
STD::Real соответствует вещественным числам
STD::String соответствует строкам

Если в качестве ключа используется имя категории, то значение соответствующее ключу имеет тип STD::Dict и в этом значении хранятся ключи и подкатегории данной категории.
